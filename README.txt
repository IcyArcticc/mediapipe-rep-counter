Program is using mediapipe model to estimate pose. It is suppossed to calculate how many squats and arm flexions were performed. It measures angles at which hands and legs are bent with arctan, based on location of 3 characteristic points on each arm and each leg. Counters can be reset via two hands straight up. It could be used e.g. to create workout app.

'q' - close camera window
'r' - reset counters
